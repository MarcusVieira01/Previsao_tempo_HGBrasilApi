Previsões do tempo atualizada a cada hora e para 10 dias

Projeto de exibição dos dados obtidos da HG Weather API, onde serão recuperados os dados de clima atual e do clima previsto para 10 dias. Foi usada a linguagem Java para requisição dos dados via HTTP Request à API, assim como tratamento, parseamento e geração dos arquivos html. Além dessa exibição, a cada atualização será remetido um e-mail cadastrado no arquivo de configuração.

O repositório do projeto consta no GitHub.
